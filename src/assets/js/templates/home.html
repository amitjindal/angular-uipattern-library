AqueApp Pattern Library App

An app for creating and managing your front-end pattern library.

View on Github

  • Easily create and manage patterns

    Create your patterns in markdown and drop the files into the AqueApp directory to update the interface.

  • Browseable interface

    Browse or search patterns by elements, components and templates. View examples, instructions and code samples.

  • Free and open source

    Make AqueApp your own. Download it on Github, install it on your machine and start adding patterns.
